# Windows 文件夹注释工具

这是一个用copilot写的 **给 Windows 文件夹添加注释（Comments）** 的 Python 工具。  
支持直接运行脚本编辑注释，也支持绑定到 **右键菜单**
支持**中文注释**

---
## 必看

1.不支持修改和删除
    如要删除/修改:
    在所注释的文件夹中, 打开隐藏文件desktop.ini, 进去修改
2.确保你打开了文件夹"备注"的显示,否则你看不到注释
